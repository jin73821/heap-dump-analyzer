Clear all results
모든 저장된 분석 결과를 삭제합니다. 힙덤프 파일은 유지됩니다.
경고
Clear All Results
모든 저장된 분석 결과가 영구적으로 삭제됩니다. 삭제 후 각 힙덤프 파일을 다시 분석해야 합니다.
⚠ This action cannot be undone.
Keep Unreachable Objects 해제
이 옵션을 해제하면 GC Root에서 도달 불가능한 객체가 분석에서 제외됩니다. 기존 분석 결과에는 영향이 없으며, 변경 사항은 다음 분석부터 적용됩니다.
Save results to disk 비활성화
이 옵션을 비활성화하면 이후 분석되는 결과가 디스크에 저장되지 않습니다.
⚠ 어플리케이션이 재시작될 경우 저장되지 않은 분석 결과는 모두 사라집니다.
Auto-analyze 활성화
업로드된 파일이 자동으로 분석됩니다. 대용량 파일 업로드 시 서버 리소스를 많이 사용할 수 있습니다.
Compress after analysis 해제
이 옵션을 해제하면 분석 완료 후 원본 덤프 파일이 압축되지 않습니다.
⚠ 대용량 힙덤프 파일이 그대로 보존되어 디스크 사용률이 크게 증가할 수 있습니다.
